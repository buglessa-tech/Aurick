⚙Installation

Upload the Aurick theme to Shopify in four steps. No technical knowledge required.

  1. Locate the theme file

    Find aurick-theme.zip on your computer. Do not unzip it.

  2. Go to Online Store → Themes

    In Shopify Admin: Admin›Online Store›Themes

  3. Upload the ZIP

    Click Add theme → Upload zip file. Select the .zip and click Upload file.

  4. Customize & Publish

    Click Customize to configure all sections in the Theme Editor, or click Publish to go live immediately.

⚠
Upload the ZIP as-is

Do not extract the file before uploading. Shopify requires the compressed format. Extracting it first will cause the upload to fail.

🔍Menus & Submenus

Shopify Navigation

Learn how to build drop-down menus, multi-level nested menus, and visual mega-menus in Shopify and integrate them into Aurick's header navigation.

Step 1: Creating Nested Menus in Shopify

Shopify supports nested navigation up to three levels deep. To create a nested structure:

  1. Go to Navigation settings

    In Shopify Admin, navigate to Admin›Online Store›Navigation.

  2. Select or Create a Menu

    Click on **Main Menu** (or click **Add menu** to create a new one).

  3. Add Menu Items

    Click **Add menu item**, name it (e.g. Collections), and link it. Then add child items below it (e.g. Chronograph, Automatic).

  4. Nest Items (Drag-and-Drop)

    Click and hold the drag handle icon next to a child menu item, drag it **up and to the right** below the parent item (e.g. drag it under Collections). This nests the item under the parent. Repeat to add sub-items under child items (up to 3 levels).

  5. Click Save Menu.

Step 2: Linking the Menu in Aurick's Header

To display your menu in the header:

  1. Open Theme Editor

    Navigate to **Online Store** → **Themes** → click **Customize** on the Aurick theme.

  2. Select the Header Section

    Click on the **Header** section in the left sidebar.

  3. Choose Menu

    Find the **Menu** setting and click **Change** → **Select menu**. Select your Main Menu and click **Save**.

Configuring Search Behavior — Search & Discovery App

  1. Open the app

    Go to Admin›Apps›Search & Discovery. If not installed, install it for free from the Shopify App Store.

  2. Set what appears in search results

    In the app click Search → Products. You can set which product fields are searched (title, description, SKU, tags, vendor, type, variants). Enable Metafield search to let shoppers find watches by case material, brand, or any other metafield you’ve defined.

  3. Configure synonyms

    Click Synonyms. Add word groups so that typing “chrono” surfaces “chronograph” watches, and “strap” finds “band” products. This dramatically improves the quality of search results for luxury vocabulary.

  4. Boost and bury products

    Click Search results → Product boosts. Pin specific products to the top of search results for target keywords — for example, boost your hero watch when a shopper searches “signature”. Similarly, bury out-of-stock items to the bottom.

ℹ
Search quality improves with more product data

Fill in your product Type, Vendor, Tags, and especially your custom Metafields on every product. The more structured data Shopify has, the more accurate and relevant the search results inside the drawer will be.

🔍Product List Filters

Collection Pages — Filter Drawer
⚠
Filters are NOT configured in the Theme Editor

The filter drawer on collection pages is powered exclusively by the free Shopify Search & Discovery app. All filter configuration — which attributes appear, their order, display type — is done inside that app, not in Aurick’s theme settings.

Why Metafield Filters Beat Tag Filters

If you filter by Product Tags (the default), a shopper browsing /collections/women sees a “Women” checkbox in the filter drawer — pointless because every product already matches. With Metafield filters, Shopify’s Search & Discovery engine automatically hides any filter value that matches the current collection context. Shoppers only see filters that are genuinely useful to narrow down their choice.

Setting Up Metafield Filters — Step by Step

  1. Create metafield definitions (once)

    Go to Admin›Settings›Metafields and metaobjects›Products›Add definition and create these definitions:

    • Brand — Namespace & key: custom.brand — Type: Single line text
    • Gender — Namespace & key: custom.gender — Type: Single line text
    • Add any others you need: Movement (custom.movement), Case Material (custom.case_material), etc.
  2. Fill in values on every product

    Open each product in Admin → scroll to the Metafields panel (at the bottom of the product editor) → enter Brand (e.g. Rolex) and Gender (e.g. Women). Click Save.

  3. Install Search & Discovery app

    If not already installed, go to Admin›Apps›Shopify Search & Discovery. It is free and made by Shopify.

  4. Configure filters in the app

    Open the app → click Filters in the left menu → click Add filter. Select your Brand and Gender metafield definitions. Choose display type: List (checkboxes) or Swatch. Remove the old Product Tag filter entry. Click Save.

✨
Result: Context-aware filter drawer

Shoppers on /collections/women see only Brand and Price. On /collections/rolex they see only Gender and Price. The collection context is hidden automatically — exactly as expected from a luxury store.

📏Size Guide

Product Page — Buy Box

Aurick includes a built-in Size Guide button on the product page. When tapped, it opens a full-width popup modal showing a size chart image for that specific product. The button hides itself automatically on products where no guide has been configured — no need to manage it manually.

ℹ
How the Theme Detects the Size Guide

The theme checks three sources in priority order. The first one found wins: (1) Product metafield custom.size_guide, (2) Variant metafield custom.size_guide, (3) Image alt text set to exactly size-guide.

Method 1 — Via Metafield (Recommended)

  1. Create the metafield definition (once)

    Go to Admin›Settings›Metafields and metaobjects›Products›Add definition. Set: Name = Size Guide, Namespace & key = custom.size_guide, Type = File → Image. Click Save.

  2. Upload the guide per product

    Open any product in Admin → scroll to the Metafields panel at the bottom → find Size Guide → upload your size chart image (JPG or PNG). Click Save product. The Size Guide button will now appear on that product page automatically.

❓FAQ

How do I change my store logo?▼

Theme Editor → Header → set Logo type to Image → upload your logo file. Recommended format: transparent PNG, 300 × 120 px.

How do I add rotating announcement messages?▼

Theme Editor → Announcement Bar → add multiple **Announcement blocks** → enable **Auto-rotate announcements**.

How do I hide a homepage section without deleting it?▼

Click the **eye icon** next to the section name inside the Theme Editor sidebar. The section becomes invisible to visitors but stays configured for future use.

⚙Watch Image → SVG Prompt

The Aurick 3D watch box hero renders your watch as a clean SVG vector inside the animated box. Paste this prompt into any AI assistant along with your watch image to get the custom SVG code:

Copy & Paste into AI (with your watch photo attached)
You are an expert SVG designer. Examine the attached watch image closely.
Your goal is to recreate ONLY the watch (the case, dial, and strap) as a clean, highly polished, responsive SVG vector file. Do NOT draw any boxes, cases, props, or backgrounds.

Follow these strict specifications:
1. Output ONLY valid, clean SVG code inside a ```xml ``` code block. Do not write any markdown explanation.
2. Background: Must be transparent.
3. Viewport: The viewBox must be exactly "0 0 300 400".
4. Vertical Alignment: The watch must be perfectly centered.
   - The center of the watch dial must sit at (x=150, y=200).
   - The strap must run straight vertically from y=10 to y=390.
5. Dimension Specs:
   - Strap Width: Exactly 60px (x=120 to x=180).
   - Outer Bezel Radius: Exactly 70px (centered at 150, 200).
   - Inner Dial Face Radius: Exactly 60px.
6. Materials & Aesthetics: Recreate case metal textures using linearGradients with stops for metallic reflections. Use radialGradients on the dial face for sunburst finishes.
7. Dial Details: Recreate hour indices, hands, minute track ticks, logos, and subdials as positioned in the reference image.
